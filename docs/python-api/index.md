---

## What to expect

Python bindings will expose the full libbitcoinkernel C API through a Pythonic interface, including:

- `ctypes` or `cffi`-based FFI layer
- Context managers for resource cleanup
- Chainstate, block, and transaction types

Follow development in the [Bitcoin Core repository](https://github.com/bitcoin/bitcoin) and community binding projects.
